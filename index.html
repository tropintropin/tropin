Туристическая компания «Пешеход Тур»

2024–2025 Написал телеграм-бота с аудиоэкскурсиями по Мончегорску для Агентства развития Мончегорска. Бот умеет запоминать маршрут и точку, на которой остановился пользователь, с помощью миниаппа показывать маршрут на карте с возможностью перехода к прослушиванию выбранной точки, показывает список рекомендаций для гостей Мончегорска с ресурса Визит.Имандра .

Stack:
Python 3.13, aiogram 3.16
Telegram Bot API 8.3
FastAPI, Leaflet JS (for Telegram Miniapp)
Redis

Развёртывание автоматизировано через команды make: с их помощью настраивается nginx, запускаются systemd-юниты для бота и миниаппа (на gunicorn), устанавливается Redis и конфигурируется окружение. nginx проксирует миниапп, бот работает как отдельный сервис, а Redis живёт как системный демон и хранит прогресс пользователей.

Реализована простая, но полезная админка: в ней можно отслеживать статистику использования — количество уникальных пользователей, процент тех, кто начал и завершил маршрут. Также через неё можно скрывать и открывать точки маршрутов, добавлять и удалять админов бота. Всё — прямо в Telegram, без лишних вмешательств в код проекта.
